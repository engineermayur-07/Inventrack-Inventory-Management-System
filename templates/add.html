How it works
Product Name Use a consistent name across all batches of the same product so the system can track them together.
Batch Number Each batch must have a unique identifier. Duplicate batch numbers are rejected automatically.
Quantity Enter the total number of units in this batch. You can use the + / − buttons or type directly.
Expiry Date Batches expiring within 30 days will trigger alerts on the dashboard. Within 10 days shows as critical.

FIFO dispatch: when you sell a product, the system automatically picks the batch with the earliest expiry date first to minimise waste.
